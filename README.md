# 🌱 Avartanā – Platform Ekonomi Sirkular Digital

Avartanā adalah aplikasi mobile berbasis UI/UX yang dirancang untuk mendukung gaya hidup berkelanjutan melalui konsep ekonomi sirkular. Aplikasi ini memudahkan masyarakat untuk menjual, menukar, mendonasikan, mendaur ulang barang bekas, serta memperoleh edukasi mengenai keberlanjutan lingkungan.

## ✨ Fitur Utama

- 🛒 Jual Barang Bekas
- 🔄 Barter Barang
- ❤️ Donasi Barang
- ♻️ Daur Ulang Barang
- 🤝 Pengajuan Mitra Daur Ulang
- 📚 Edukasi Sirkular
- 📦 Detail Barang
- 🚚 Pengiriman
- 💳 Pembayaran
- 📊 Progress Donasi
- 👤 Profil Pengguna

## 🎯 Tujuan

- Mengurangi limbah melalui pemanfaatan kembali barang layak pakai.
- Mendukung penerapan ekonomi sirkular berbasis komunitas.
- Mendorong gaya hidup yang lebih ramah lingkungan.
- Menyediakan platform digital yang mudah digunakan.
